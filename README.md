# 🤖 AI Personal Assistant

A smart, fast, and web-based AI Personal Assistant built using **Python, Flask**, and **Groq's Llama 3 API**. This application can answer general questions and perfectly summarize long emails in seconds.

## 🚀 Live Demo
**[Click here to use the AI Assistant](https://ai-personal-assistant-2-tl0h.onrender.com)**

## ✨ Features
* **Ask Anything:** Get instant, smart answers to any question.
* **Email Summarizer:** Paste a long email and get a crisp 2-3 sentence summary.
* **Fast Responses:** Powered by Groq's lightning-fast inference engine.
* **Responsive UI:** Clean and simple web interface.

## 🛠️ Tech Stack
* **Backend:** Python, Flask
* **Frontend:** HTML, CSS, JavaScript
* **AI Model:** Llama-3.1-8b-instant (via Groq API)
* **Deployment:** Render
